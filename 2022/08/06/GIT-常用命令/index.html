GIT 基本操作

说明：

  • workspace：工作区
  • staging area：暂存区
  • local repository：本地仓库
  • remote repository：远程仓库

一般工作流程

  1. 将项目拷贝到本地

    git clone [url]

    拉取远程仓库的新提交：

    git fetch  // 不会自动合并分支
git pull  // 拉取远程仓库上当前分支的最新提交，并合并到本地分支上
git pull [remote-shortname] [local-branch]

  2. 创建自己的工作分支

    git branch [new-branch-name]  // 创建分支
git checkout [new-branch-name]  // 切换到指定分支

    或：

    git checkout -b [new-branch-name]  // 创建并切换到新分支

  3. git commit
